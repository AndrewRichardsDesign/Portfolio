Splitview Column Widths

To create a consistent experience across all iPad sizes with appropriately proportioned column widths, we developed two rules that mix fixed width and percentage based width.

  • If the screen width = 1145 points or less, the left column = 382 points.
  • If the screen width = 1146 points or more, the left column = 1/3 of the screen width.
